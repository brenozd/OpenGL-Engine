# OpenGL Engine
____

### What is this?
This is an informal study about C++ 11 and OpenGL.

Keep in mind that this isn't a professional project and has learning purposes only!

### What was used?
In this project we used:
* C++ 11 and g++ 9.3.0
* OpenGL 4
* GLFW 3
